# CFB Play-Call Tendency Analyzer

**Live demo:** [cfb-playcall-tendencies.streamlit.app](https://cfb-playcall-tendencies.streamlit.app)

A Streamlit web app for analyzing college football offensive play-call tendencies using live data from the [College Football Data API](https://collegefootballdata.com).

## What It Does

Select any Power 5 team and season(s), apply situational filters, and instantly see:

- **Run/Pass Split** — overall percentages and yards-per-play
- **Play Type Breakdown** — frequency chart of every play type used
- **Field Zone Tendencies** — run vs. pass split across 5 field zones
- **Top Situational Tendencies** — most common (down, distance, field zone, play type) combinations
- **AI Scouting Summary** — 3-4 sentence plain-English report generated by an LLM
- **CSV Export** — download filtered plays for your own analysis

## Getting Started

### You'll Need

1. A free **CFBD API token** — get one at [collegefootballdata.com/key](https://collegefootballdata.com/key)
2. *(Optional)* A free **Groq API key** for AI scouting summaries — get one at [console.groq.com](https://console.groq.com)

### Run Locally

```bash
git clone https://github.com/adam-shams/CFB-Play-Tendencies.git
cd CFB-Play-Tendencies
python -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```

Enter your CFBD API token in the sidebar and click **Analyze**.

## Filters

| Filter | Description |
|---|---|
| Team | Any Power 5 school |
| Seasons | 2021–2025 (multi-select) |
| Down | 1st through 4th |
| Distance | Yards to first down range |
| Yards to Goal | Field position range (0 = goal line, 99 = own 1-yd line) |
| Score Situation | 9 buckets from Blowout Loss to Blowout Lead |
| Quarters | Filter to specific quarters |

## Field Zones

| Zone | Yards to Goal | Field Position |
|---|---|---|
| Pinned Deep | 76–99 | Own 1–24 yard line |
| Own Territory | 51–75 | Own 25–49 yard line |
| Midfield | 40–50 | Around midfield |
| Opp Territory | 21–39 | Opponent's 21–39 yard line |
| Red Zone | 0–20 | Inside opponent's 20 |

## AI Scouting Summary

The app supports two LLM backends (configured in the sidebar under ⚙️ AI Summary Settings):

- **Groq (default)** — cloud-based, free tier, uses `llama-3.1-8b-instant`
- **Ollama** — local inference, requires [Ollama](https://ollama.com) running with a model pulled (e.g. `ollama pull llama3.2`)

## Architecture

```
cfb_tendency_analyzer/
├── main.py               # Streamlit UI only
├── mcp_tools/
│   ├── fetch_plays.py    # CFBD API calls + local JSON caching
│   ├── filter_plays.py   # Situational filtering
│   ├── analyze_plays.py  # Run/pass analysis, field zones, tendencies
│   └── summarize.py      # LLM scouting summary (Groq / Ollama)
└── utils/
    ├── constants.py      # Field zones, score buckets, play type sets
    └── helpers.py        # classify_play(), normalize_play(), etc.
```

Play data is cached after the first fetch, so re-running filters on the same team/season is instant.
